RENESSEANCE
Admin Bot

Комплексная административная система для игрового клана. Полный стек управления — от заявок на вступление до уведомлений о выбросах в реальном времени.

Python 3.11 discord.py SQLite Slash Commands Modals Webhooks
Features

Функциональность бота

СИСТЕМА ЗАЯВОК

Приём и обработка заявок на вступление в клан через интерактивные модальные окна.

  • Подача заявки через модальное окно
  • Автоматическая публикация в канал для рассмотрения заявок
  • Рассмотрения заявки кнопками
  • Уведомление заявителя о решении

МОДЕРАЦИЯ ЗАЯВОК

Полная панель управления заявками для офицеров и командиров клана.

  • Список активных заявок
  • Одобрение / отклонение / вызов на обзвон с причиной
  • История рассмотренных заявок
  • Логирование действий модераторов

СИСТЕМЫ СБОРОВ

Организация кланового сбора — расписание, отметка присутствия, статистика явки.

  • Создание сбора с датой и целью
  • Кнопка отметки «Буду»
  • Список участников в реальном времени
  • Напоминания по выставленным настройкам(дата, время)

УЧЁТ ИГРОКОВ КЛАНА

База данных участников с ролями, датой вступления и активностью.

  • Иерархия ролей клана
  • Распределение игроков по иерарзии
  • Обновление состава каждые 30 минут

УВЕДОМЛЕНИЯ О ВЫБРОСАХ

Включение и отключение уведомлений о выбросах

  • Авто-пинг роли при выбросе
  • Включение и отключение уведомлений о выбросах

EMBED КОНСТРУКТОР

Создание красивых embed-сообщений через интуитивный интерфейс без кода.

  • Заголовок, описание, цвет
  • Поля и значения
  • Изображение и футер
  • Превью перед публикацией
